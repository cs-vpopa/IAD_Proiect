# IAD_Proiect

This repository contains a Flask application for seed clustering. 

## Contents
- `iad_app.py`: Main application code.
- `templates/index.html`: HTML template for the web interface.
- `model.pkl`: Pre-trained model for seed clustering.
- `seeds.csv`: Dataset used for clustering.

## Instructions
1. Clone the repository.
2. Navigate to the directory.
3. Run the Flask application:
   ```bash
   python iad_app.py
